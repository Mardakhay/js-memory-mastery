# JS Memory Mastery

An interactive deep-dive into how JavaScript manages memory under the hood — built as a single, self-contained HTML file with no dependencies.

## What's Inside

15 sections covering every layer of JS memory, each with live simulations and animations:

- Memory Life Cycle (Allocation → Usage → Release)
- Stack Memory — LIFO push/pop demo
- Heap Memory — dual-panel stack + heap visualizer
- Call Stack — step-by-step function execution walker
- Stack Overflow — animated infinite recursion simulator
- Primitives vs References — copy and mutation behavior
- Shallow vs Deep Copy — side-by-side object mutation demo
- `const` behavior — why objects can still mutate
- String Pool — interning vs `new String()` visualized
- Garbage Collection — interactive Mark & Sweep simulator
- V8 Engine Pipeline — Parse → Ignition → TurboFan → Machine Code
- Closures, Memory Addresses, Memory Limits
- 8-question quiz with instant feedback

## Usage

Download `js-memory-mastery.html` and open it in any modern browser. No installation, no server, no internet connection required.

## Tech

Pure HTML + CSS + Vanilla JavaScript. Zero libraries. Zero dependencies.
